## 🔬 Featured Projects

### 1) Computational Design of RNA Thermoswitches for High-Temperature Genetic Control in *Bacillus subtilis*

**MSc Thesis | Machine Learning + Computational Biology + Thermodynamic Modelling**

#### Overview
This project focused on designing RNA thermoswitches for **high-temperature genetic control** using computational modelling and machine learning. It combined biological sequence design, thermodynamic simulation, and predictive modelling to identify promising RNA candidates in a reproducible way.

#### Why it matters
This work shows how **data-driven methods can support biological design**, helping reduce large experimental search spaces before lab validation and making early-stage screening more efficient.

#### What I did
- Built a computational workflow for RNA candidate generation and screening
- Used **NUPACK** for thermodynamic and structural simulation
- Engineered sequence-based and structure-informed features
- Applied machine learning to rank and prioritise candidate sequences
- Organised the work as a reproducible research pipeline

#### Tools & methods
- Python
- pandas
- NumPy
- scikit-learn
- NUPACK
- Feature engineering
- Predictive modelling
- Reproducible workflow design

#### Key results
- Achieved **ROC-AUC of 0.951** and **PR-AUC of 0.875** on the test split
- Built a ranking workflow for prioritising biologically promising RNA candidates
- Combined thermodynamic simulation with machine learning in an interdisciplinary research setting

🔗 **Repository:** https://github.com/mdabir-hossain/rna-thermoswitch-ml

### 3) Sepsis Early Warning System

**Healthcare Machine Learning Project | Early Risk Prediction + Calibration + Decision Support**

#### Overview
This project focused on early sepsis prediction using ICU patient data. The goal was to build a machine learning workflow that reflects a more realistic healthcare setting, where **calibration, threshold selection, and decision trade-offs** matter alongside predictive performance.

#### Why it matters
Sepsis is a serious medical condition where earlier risk detection can support faster intervention. In healthcare, models need to be not only accurate, but also **interpretable, reliable, and evaluated in a decision-aware way**.

#### What I did
- Built a structured machine learning pipeline for sepsis prediction
- Processed and prepared clinical time-series data
- Performed feature engineering from physiological measurements
- Compared baseline and improved models
- Evaluated both discrimination and calibration performance
- Produced threshold analysis and alert trade-off outputs
- Created explainability and reproducibility artifacts for interpretation

#### Tools & methods
- Python
- pandas
- NumPy
- scikit-learn
- Matplotlib
- Calibration analysis
- Threshold tuning
- Explainability workflow
- Reproducible project structuring

#### Key results
- Built an end-to-end early warning pipeline for ICU sepsis prediction
- Produced calibration, threshold, and alert trade-off analyses for decision-focused evaluation
- Generated explainability and reproducibility artifacts to support transparent model assessment
